## 1. Objective

The objective of this lab was to gain hands-on experience with Azure Blob Storage by hosting a static website using Azure's Static Website feature. This involved creating a cloud storage account, enabling static website hosting, building a simple HTML site locally in Visual Studio Code, and deploying it to Azure using the Azure Storage extension. The lab also demonstrated how static website hosting on Blob Storage differs from more complex hosting solutions like Azure App Service, highlighting its simplicity and cost-effectiveness for serving content that requires no server-side processing.
